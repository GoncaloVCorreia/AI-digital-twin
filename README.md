## How to Run the Project

1. Install dependencies using [uv](https://github.com/astral-sh/uv):
   ```bash
   pip install uv
   ```

2. Sync the environment:
   ```bash
   uv sync
   ```

3. Navigate to the `src` directory:
   ```bash
   cd src
   ```

4. Run the application:
   ```bash
   uv run main.py
   ```
